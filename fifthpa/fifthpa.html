Why Javascript?

This is why

（一）上課內容

    W1:

  • 書寫模式
  • Variables
  • Data type

    W2:

  • Function
  • Function 練習們：上課時老師會示範一些應用Function 的程式方式，我會和老師一起寫這些程式，一方面幫我熟悉程式功能的書寫方式，另外也可以讓我見識這些code可以做的事情的多元性。以下為其中兩個例子，使用的網站為codepen，一樣為課程老師推薦的網站，這個網站會很快的更新，使我可以一邊寫程式，一邊看看自己的程式有沒有正確，我覺得很適合練習的時候使用。

    W3:

  • Array: 可以一次放很多物件
  • Loops: While
  • Conditions: If else

    W4:

  • Forms: 練習製作表單：checkboxs, radio box

（三）課堂實作

實作一：第二週課程project

此project為寫程式將滑鼠移到網站下方的三張圖片後可以在上方紫色區塊顯示滑到那張圖的alt text

實作二：第四週課堂project

這個project的目標為設計一個程式，讓填表單的人若按下"Is the billing message the same前的勾<，程式會自動幫下面兩欄填入相同的資訊

（三）過程＆心得

javascript 對我比較抽象 有些function一開始也不知道要怎麼設項最後一個project就在腦袋停留了兩天 但做出來的效果讓網頁更精緻！
